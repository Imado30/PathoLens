PathoLens Platform Tutorial

Step 1 of 18

1. Login to PathoLens

Begin by accessing the PathoLens login page. Enter your email and password to access your account.

Note: If you don't have an account yet, click "Create your account" to register first.

2. Creating an Account

To create a new account, fill in your:

  • First and Last Name
  • Email address
  • Password (enter it twice to confirm)

3. Homepage Overview

After logging in, you'll see your dashboard with these options:

  • Continue Diagnosis: Resume an unfinished diagnosis
  • New Diagnosis: Start a new case

4. Accessing the Help Guide

The Help Guide provides answers and instructions for using PathoLens. You can access it from the navigation menu.

5. Starting a New Diagnosis

Click "New Diagnosis" to begin. You'll need to select one of the available datasets:

  • Each dataset contains different subjects
  • The system will randomly select a subject from your chosen dataset

6. Diagnosis Page Interface

The diagnosis page shows three views of a 3D MRI/CT scan from different angles.

Navigation Tip: Left-click and drag on one image to scroll through layers in all three views simultaneously.

7. Switching Between Image Formats

You can toggle between different image types:

  • FLAIR: Fluid-attenuated inversion recovery
  • T1: T1-weighted imaging

Use the buttons at the bottom of the screen to switch views.

8. Drawing a 3D Rectangle

To mark lesions:

  1. Draw the first 2D rectangle
  2. Click "Jump Rectangle"
  3. Draw the second rectangle in another view

9. Setting Confidence Levels

After drawing each lesion, you'll be prompted to:

  • Save the lesion
  • Select your confidence level (1-10)

10. Managing Drawn Lesions

You can review and remove lesions:

  • Use the "Show Lesions" overlay to view all marked lesions
  • Click "X" to remove unwanted lesions

11. Submitting Final Confidence

When finished with all lesions:

  1. Provide your overall confidence level
  2. Click "Confirm" to submit

12. AI Analysis Comparison

The AI page shows:

  • Your diagnosis
  • AI's analysis (from multiple models)
  • Option to show overlay comparisons

13. Selecting Final Diagnosis

Choose between:

  • Your original diagnosis
  • The AI's diagnosis
  • Or edit your diagnosis further

14. Editing Your Diagnosis

The edit page allows you to:

  • Modify your previous diagnosis
  • View AI and your diagnoses side-by-side
  • Use zoom features for detailed comparison

15. Zoom Feature

Use the zoom feature to:

  • View your previous diagnosis in detail
  • Compare it with the AI's diagnosis

16. Lesion Overlay View

This view shows:

  • All edited lesions
  • Options to restore original lesions
  • Confidence levels for each lesion

17. After Diagnosis Completion

After finishing, you can:

  • Start a new diagnosis immediately
  • Return to the homepage

18. Continuing Unfinished Work

If you have an unfinished diagnosis:

  • The system will remind you when you log back in
  • Click "Continue Diagnosis" to resume where you left off
  • You must complete current diagnosis before starting new ones

19. Tutorial Complete!

You've now learned all the essential features of PathoLens.

Remember: You can always access the Help Guide from the navigation menu if you need to review any features.

Happy diagnosing!
